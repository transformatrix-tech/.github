# Organization-wide community files

This repository contains organization-level templates and guidance used across repositories under this organization.

- ISSUE_TEMPLATE/ — issue templates (bug/feature)
- pull_request_template.md — PR template used across repos
- CONTRIBUTING.md — how to contribute
- CODE_OF_CONDUCT.md — expected behaviour
- SECURITY.md — how to report security issues
- CODEOWNERS — default code owners

If you need a new template or want to change default behaviour, open an issue here and tag @admins.
